# Руководство пользователя BI-системы

## Общее описание

BI-система объединяет данные из amoCRM, 1C и Google Таблиц в единое хранилище и строит дашборды для руководства.

## Доступы

- **Metabase (дашборды):** http://SERVER_IP:3000
- **API (ввод данных):** http://SERVER_IP:8000/docs
- **PostgreSQL:** localhost:5432 (только для админов/разработчиков)

## Роли

| Роль | Что видит |
|---|---|
| CEO / Собственник | Все дашборды |
| CFO / Главбух | Финансовые дашборды (P&L, Cashflow, расходы) |
| РОП | Воронки продаж, KPI менеджеров |
| Линейный сотрудник | Свои показатели + форма ввода данных |

## Как войти

1. Откройте Metabase по адресу.
2. Введите логин и пароль (выдает администратор).
3. Выберите дашборд "Финансовый" или "Коммерческий".

## Как внести данные вручную

1. Откройте http://SERVER_IP:8000/docs (Swagger UI).
2. Авторизуйтесь: нажмите `/auth/token` → "Try it out", введите логин/пароль.
3. Скопируйте полученный `access_token`.
4. Нажмите кнопку "Authorize" вверху страницы и вставьте токен.
5. Перейдите в раздел `/input/expenses` (или `/input/transactions`).
6. Заполните поля и отправьте. Система автоматически:
   - проверит типы данных,
   - конвертирует валюту в рубли,
   - запишет в журнал аудита.

## Если что-то не так

- **Данные не обновились:** ETL запускается 2 раза в день. Подождите до следующего запуска (07:00 и 13:00 по МСК).
- **Неверная сумма в валюте:** Проверьте, что курс валюты загружен в разделе `fact_exchange_rates`.
- **Ошибка доступа:** Обратитесь к администратору для проверки роли.
